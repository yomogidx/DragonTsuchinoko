# DragonTsuchinoko

![gif](https://github.com/yomogidx/DragonTsuchinoko/blob/main/image/movie.gif)

## ゲーム概要（Overview）

ドラゴン形態のつちのこちゃんを動かしてほこりを掃除するゲームです。

## 操作方法

・移動：キーボード（W/A/S/D）or クリック
